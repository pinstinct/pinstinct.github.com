## Box Model

- margin : outside of the border
- border
- padding : within the border
- (html element)

## Selectors

### 1. element selector
```css
p {
	text-align: center;
	color: red;
}
```

### 2. id selector

```css
#uniquemyid {
	text-align: center;
	color: red;
}
```

### 3. class selector
```css
.jumbotron {
	text-align: center;
	color: red;
}
```

## ID vs Class
- ID : **unique**, link to take a user to that section.
- IDs are more specific than classes so they will take priority.
