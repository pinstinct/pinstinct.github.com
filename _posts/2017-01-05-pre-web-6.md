# Software Engineering

## 개발 vs 구현

- 개발 : 분석, 디자인, 코딩, 테스트까지의 과정 (like front-end 개발자)
- 구현 : 개발한 SW를 시스템에 맞춰 설치하는 것 (like 웹퍼블리셔)

## Trend of Software Engineering
